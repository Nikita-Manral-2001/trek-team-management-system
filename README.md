# 🏔️ Trek Itinerary & Team Management System

## 📌 Overview

This project is a complete Trek Itinerary & Team Management System developed using ASP.NET MVC.

It is designed to manage trek planning, team assignments, and operational workflows. The system enables operations teams to efficiently organize treks, assign staff, and track execution with a structured approval process.

---

## 🚀 Features

### 🧭 Trek Itinerary Management

* Create **Custom Trek Itinerary**
* Create **Fixed Departure Itinerary**
* Add dynamic trek days
* Manage start & end dates
* Edit and delete itineraries

---

### 📅 Trek Listing

* View all treks with:

  * Trek Name
  * Start Date / End Date
  * Team required dates
  * Itinerary type (Fixed / Customize)
* Edit/Delete trek itineraries

---

### 👥 Team Assignment

* Assign team members:

  * Trek Leader (TL)
  * Base Camp Manager (BCM)
  * Chef
* Assign team based on trek schedule

---

### 📊 Date-wise Trek Management

* View treks **date-wise**
* Track:

  * Which trek is running on which date
  * Assigned team members
* Helps in resource planning

---

### 📝 Team Form Workflow

* When a team member is assigned:

  * A form is automatically sent to their dashboard
* Team fills details like:

  * Number of trekkers
  * Mules / Porters
  * Trek execution details
  * Completion status

---

### ✅ Approval System

* Only **Operation Manager** can approve forms

Workflow:

1. Team member fills form
2. Form goes to Operation Manager
3. Manager reviews & approves

---

### 📂 Data Views

#### 👤 Employee Data

* Track:

  * Assigned treks
  * Team availability

#### 🔤 Trek-wise View

* View all records grouped by trek name

#### 📆 Departure-wise View

* View treks based on departure dates
